모델 아키텍처

🔍

CNN 레이어

오디오 데이터를 처리하여 로컬 특징을 추출합니다. Conv1d와 MaxPool1d를 사용하여 주파수 패턴을 학습합니다.

nn.Conv1d(input_size, 128, kernel_size=3)
nn.MaxPool1d(kernel_size=2)
🔄

Transformer 레이어

CNN 결과를 시퀀스 형태로 변환하여 시간적 의존성을 학습합니다. 복잡한 패턴을 효과적으로 포착합니다.

nn.TransformerEncoderLayer(
  d_model=32, nhead=2
)
📊

분류 레이어

최종 출력으로 비명 여부를 판단하는 이진 분류를 수행합니다. Sigmoid 활성화 함수를 사용합니다.

nn.Linear(32, num_classes)
torch.sigmoid(output)
